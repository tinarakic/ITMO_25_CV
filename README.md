# Проект - Подсчет количества людей в магазине

В данном репозитории представлен код проекта по дисциплине "Проектирование нейроных сетей для задач анализа изображений" (ИТМО, 2025) на тему "Подсчет количества людей в магазине".

## Цель проекта
Цель проекта — обучить модель компьютерного зрения для обнаружения людей в магазине и подсчёта их количества на основе изображений взятых из камер видеонаблюдения из магазина.

В качестве исходных данных используется датасет
Consumers Bid Dataset: https://www.kaggle.com/datasets/angelosgiotis/consumers-bid

## Описание решения

В рамках проекта:
- используется модель детекции объектов (Faster R-CNN с бэкбоном ResNet18)
- выполняется обнаружение людей на изображениях
- рассчитывается общее количество людей в кадре
- оценивается качество модели с помощью стандартных метрик (mAP, mIoU)

## Результат

Модель способна находить людей на изображениях магазина и считать их количество.
